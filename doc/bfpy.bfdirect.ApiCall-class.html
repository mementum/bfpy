__call__(self, instance, request)
(Call operator)

source code 

Invoked as method by ApiService instances. It receives the instance and a request

It finishes preparation of the soap message with the value of the request object and also the preparation of the http request.

Invokes the http request, the parsing of the soap and return the "Result" part of the entire response

Parameters:
  • instance (an object) - object that is invoking the descriptor
  • request (ApiDataType) - the request to be sent to the servers
