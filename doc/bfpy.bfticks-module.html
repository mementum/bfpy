GetPriceTicks(price, ticks, betType)

source code 

Returns the next price tick upwards/downwards for a given price and bet type

Back bets increase price upwards Lay bets increase price downwards

Parameters:
  • price (float) - base price for next tick
  • ticks (int) - number of ticks to move the price
  • betType (B|L) - bet type (determines the direction)
Returns: float
the passed prices decreased x ticks

GetClosestTick(price, down)

source code 

Returns the next (and best) valid price upwards/downwards given a price (which may or may not be valid) It is of course limited to PriceMin/PriceMax

Parameters:
  • price (float) - base price for next price
  • down (bool) - go downwards (back bets) if True, else go upwards (lay bets)
Returns: float
the best price upwards/downwards
