BfPy main module. It imports the main objects to let them be re-imported

  • bfpy.bfapicalls: Definition of direct construction API call and services and metaclass to install them in an API provider
  • bfpy.bfclient: BfClient object implementation.
  • bfpy.bfclientext: BfApi method extensions to be inserted in BfClient
  • bfpy.bfcounter: Implementation of a Data Counter/Throttler to implemente Betfair Data Charges weight counting
  • bfpy.bfdirect: Definition of direct construction API call and services and metaclass to install them in an API provider
  • bfpy.bferror: BfPy exceptions
  • bfpy.bfglobals: BfPy global variables and functions module.
  • bfpy.bfprocessors: Implementation of BfApi request and response processors
  • bfpy.bfservice: Definition of services and metaclass to install them in an API provider
  • bfpy.bfsoap: Soap processing
  • bfpy.bftransport: BfPy transport
  • bfpy.bftypes: Classes for definition and management of API types
  • bfpy.bfwsdl: BfPy wsdsl variables holding the Betfair WSDL definitions
  • bfpy.timezone: BfPy Timezone classes.
  • bfpy.util: BfPy utility classes
